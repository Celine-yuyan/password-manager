# password-manager
password-manager is a console program that stores and generates passwords
# Getting started
1. Install python
2. Clone the respository by running git clone https://github.com/Celine-yuyan/password-manager.git. Or you can download it as a .zip folder then extract it
3. Install the python modules with pip
4. Run password_manager.py
